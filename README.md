# Kvadratická rovnice (CLI)

Jednoduchá konzolová aplikace pro řešení kvadratických rovnic.

## Použití
python cli.py --a 1 --b -3 --c 2

## Parametry
--a, --b, --c : koeficienty rovnice  
--help        : nápověda  
--version     : verze programu
